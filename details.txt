# AI Support Agent

## WhatsApp + AI + Ticketing + Appointment Booking System

---

# 1. Project Overview

The AI Support Agent is an intelligent customer support automation platform that allows customers to communicate through WhatsApp and receive instant support.

The system can:

* Answer customer queries
* Create support tickets
* Register complaints
* Book appointments
* Track ticket status
* Store customer information
* Provide AI-powered support
* Manage all activities through an Admin Dashboard

The solution combines:

* WhatsApp Cloud API
* FastAPI
* Ollama LLM
* SQLite Database
* Google Calendar
* LangGraph (future)
* RAG Knowledge Base (future)

# 17. Common Problems & Fixes

---

## Problem 1

### Error

```text
Authentication Error
Code 190
```

### Cause

Token expired.

### Fix

Generate new token from Meta.

Update:

```env
WHATSAPP_TOKEN=
```

Restart server.

---

## Problem 2

### Error

```text
Verification failed
```

### Cause

Wrong verify token.

### Fix

Ensure:

```env
VERIFY_TOKEN=
```

Matches Meta dashboard.

---

## Problem 3

### Error

```text
TemplateNotFound
```

### Cause

Missing HTML file.

### Fix

Ensure:

```text
templates/
```

Contains all templates.

---

## Problem 4

### Error

```text
Connection Refused
```

### Cause

FastAPI not running.

### Fix

Run:

```powershell
uvicorn whatsapp:app --reload
```

---

## Problem 5

### Error

```text
Ollama Timeout
```

### Cause

Model not running.

### Fix

Run:

```powershell
ollama run llama3
```

---

## Problem 6

### Error

```text
Ticket not found
```

### Cause

Wrong Ticket ID.

### Fix

Verify ticket exists in database.
